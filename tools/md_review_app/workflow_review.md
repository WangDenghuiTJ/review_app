# Demo Review Document

Last updated: 2026-04-10

## Purpose

This file is a public demo document bundled with the app source tree.

Use it when you want to test:

- loading a Markdown file into Review App
- adding comment threads
- switching between clean mode and revision mode
- exporting the current document as PDF

## Demo Checklist

1. Open the document in the browser.
2. Add a comment on one sentence.
3. Reply in the same thread.
4. Ask the agent to revise a sentence directly.
5. Verify that revision marks appear in review mode.
